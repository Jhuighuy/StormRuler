Detailed Description

Permutation utilities.

Permutation can be used as a mapping from the permuted to the unpermuted indices of a collection. Common usage pattern:

std::vector<T> values = {...};
std::vector<T> permuted_values(values.size());
for (size_t i = 0; i < permuted_values.size(); ++i) {
permuted_values[i] = values[permutation[i]];
}
